# Anime

Сайт по аниме на чистом JavaScript

## 📖 Описание

Веб-приложение для просмотра информации об аниме с каталогом, поиском и детальными страницами. Проект использует vanilla JavaScript без фреймворков.

## ✨ Функционал

- **Главная страница** — слайдер с популярными аниме и список топ просмотров
- **Каталог** — просмотр аниме по категориям
- **Поиск** — поиск по названию аниме
- **Детальная страница** — полная информация об аниме (описание, жанры, рейтинг, теги)
- **Авторизация** — форма входа (login page)
- **Адаптивное меню** — мобильная навигация
- **Кнопка "Наверх"** — плавная прокрутка к началу страницы
- **Модальные окна** — для взаимодействия с пользователем

## 🛠 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — стилизация, анимации
- **Bootstrap** — адаптивная сетка
- **JavaScript (ES6+)** — логика приложения
- **Swiper.js** — слайдер для главной страницы
- **Font Awesome & Elegant Icons** — иконки
- **JSON** — хранение данных (db.json)

## 🚀 Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# Или через Node.js (npx)
npx serve
```

## 📊 Данные

Проект использует `db.json` с информацией об аниме:
- Название (русское и оригинальное)
- Жанр и теги
- Год выпуска
- Рейтинг и количество просмотров
- Описание сюжета
- Ссылка на изображение

## 🎨 Особенности

- Чистый код без зависимостей от тяжелых фреймворков
- Быстрая загрузка страниц
- Responsive дизайн для всех устройств
- Интеграция с внешними API (изображения с MyAnimeList/Shikimori)
